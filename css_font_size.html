Resize the browser window to see how tha text size scales

Use the "vw" unit when sizing the text. 10vw will set the size to 10% of the viewport width.

Viewport is the browser size. 1vw= 1% of viewport width. If the viewport is 50cm wide, 1vw id 0.5cm
